# UI-UX-Experiment 5

## Aim:

To identify user experience problems and propose UX improvements for two digital platforms.

## Algorithm:

1. Analyze user reviews and feedback.
2. Identify common user pain points.
3. Define UX problem statements.
4. Create user personas.
5. Suggest design improvements.

---

## Scenario 1: Designing a Health App Platform

### Pain Point:

Patients find it difficult to track and monitor health progress

### Problem Statement:

Users need a clear way to monitor health progress because poor visibility reduces motivation and engagement.
